# Training Planner

**Автор:** Лысенко Валерий

## Описание

Training Planner - это десктопное GUI-приложение для планирования и учета тренировок. Приложение позволяет добавлять, удалять и фильтровать тренировки, а также сохранять данные в JSON-файл.

### Возможности

- Добавление новых тренировок с указанием даты, типа и длительности
- Валидация вводимых данных (формат даты, положительная длительность)
- Фильтрация тренировок по типу и дате
- Удаление записей
- Автоматическое сохранение и загрузка данных в JSON
- Удобный табличный интерфейс
- Сортировка записей по дате

## Технологии

- **Python 3.x**
- **Tkinter** - для создания графического интерфейса
- **JSON** - для хранения данных
- **Git** - для контроля версий
